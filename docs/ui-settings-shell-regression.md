# 设置壳层 UI 回归基线（主页同页设置）

## 目标
- 保证主页与设置页在同一窗口切换时，左栏宽度、背景、分隔、圆角、头部安全区一致。

## 手工回归步骤
1. 启动应用并进入主页。
2. 记录左栏宽度，拖拽分隔条改变宽度。
3. 打开设置（`#/settings`），检查左栏宽度是否继承上一步结果。
4. 在设置页点击“返回应用”，检查宽度是否保持不变。
5. 检查主页/设置左栏背景色是否一致（同一主题下无明显跳变）。
6. 检查右侧内容区与左栏交接处：左上角、左下角圆角半径一致。
7. 检查 mac 红绿灯区域：设置页“返回应用”不被遮挡。
8. 切换 `providers`、`coder`、`general`，确认右侧均有统一标题栏。

## 验收标准
- 左栏宽度来源唯一（`--app-left-pane-width`），主页与设置共用。
- 左栏壳层来源唯一（`AppShellLeftPane`），无页面级重复补丁逻辑。
- 设置页“返回应用”按钮与主页头部按钮共享尺寸/圆角 token。
- 切换页面时无明显布局跳变、无分隔线双重叠加、无圆角不一致。
