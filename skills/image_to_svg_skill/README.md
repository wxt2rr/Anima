# Image to SVG Skill Package

This package lets an agent convert a reference image into SVG outputs.

It supports two practical outputs:

- `exact.svg`: pixel-perfect visual reproduction by embedding the original image.
- `editable.svg`: approximate editable vector SVG generated by color tracing.

For diagrams, charts, and PPT-style graphics, use the generated editable SVG as a starting point, then manually refine the paths and replace traced text with real `<text>` elements.

## Install Dependencies

```bash
pip install pillow numpy opencv-python svgwrite
```

Optional, for SVG-to-PNG rendering and diff workflows:

```bash
pip install cairosvg
```

## Usage

```bash
python scripts/image_to_svg.py input.png --out output --mode both --preview
```

Outputs:

```text
output/
  exact.svg
  editable.svg
  preview.html
  source.png
```

## Modes

```bash
# Pixel-perfect visual SVG, image embedded as base64
python scripts/image_to_svg.py input.png --out output --mode embed

# Editable vector approximation
python scripts/image_to_svg.py input.png --out output --mode trace --colors 10

# Both outputs
python scripts/image_to_svg.py input.png --out output --mode both
```

## Important Note

A PNG/JPG screenshot cannot be converted into a perfectly editable SVG automatically, because the original vector data is gone. This skill therefore separates visual exactness from editability.
