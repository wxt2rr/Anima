# 生成流程检查清单

## 阶段 1：需求确认

- [ ] 是否明确视频目标与受众。
- [ ] 是否明确输出规格（时长/画幅/分辨率）。
- [ ] 是否明确风格限制与禁用项。
- [ ] 是否确认工具模式（文生视频/参考图生视频/图生视频）。

## 阶段 2：创意产出

- [ ] 是否给出 3 个可区分的创意方向。
- [ ] 每个方向是否包含画面母题与叙事骨架。
- [ ] 是否有默认兜底方向（平衡）。

## 阶段 3：Prompt 生成

- [ ] 主 prompt 是否覆盖主体、动作、场景、镜头、风格。
- [ ] 是否提供至少 3 条负面约束。
- [ ] 参数建议是否与目标节奏一致。

## 阶段 4：迭代改写

- [ ] 是否先描述失败现象，再给改写。
- [ ] 是否每轮只改一类变量。
- [ ] 是否保留上一版可回退记录。
- [ ] 连续两轮失败时是否执行回退或换方向。
